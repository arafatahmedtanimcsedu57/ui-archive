# Button Component

The `Button` component is a versatile UI element with various styles, sizes,
and support for icons.

## Default Buttons

Basic button styles include `Primary`.

<Canvas>
  <Story name="Default Buttons">
    <div className="flex flex-wrap items-center gap-5">
      <Button>Primary Button</Button>
    </div>
  </Story>
</Canvas>

```jsx
<Button>Primary Button</Button>
```

## Button Sizes

The Button component supports different sizes: Small, Medium, and Large.

<Canvas>
  <Story name="Button Sizes">
    <div className="flex flex-wrap gap-5 items-center">
      <Button size="sm">Small Button</Button>
      <Button>Medium Button</Button>
      <Button size="lg">Large Button</Button>
    </div>
  </Story>
</Canvas>

```jsx
<Button size="sm">Small Button</Button>

<Button>Medium Button</Button>

<Button size="lg">Large Button</Button>
```

